# atp-website

To install dependencies:

```bash
bun install
```

To run the shop from the project root:

```bash
bun run shop
```

The shop is served at `http://localhost:3003`.

To run the API, client, admin, and shop together:

```bash
bun dev
```

This project was created using `bun init` in bun v1.1.27. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
